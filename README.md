# Python Class

## Python class homework and modules.

This GitHub project contains all the code for modules and assignments.  It also contains
code for examples and playgrounds for miscellaneous modules.

## Virtual Environment

This project uses external modules and has a virtual environment configured for easy use.
To load the virtual environment, start at the top folder and run the following in your terminal:

`venv\Scripts\activate`
